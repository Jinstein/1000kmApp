# 1000km 걷기 챌린지 앱

1년 동안 누적 1000km 걷기 목표를 달성하기 위한 iOS 앱입니다.

## 기능

- 하루 걸은 거리(km) 입력 및 기록
- 누적 거리 및 남은 거리 실시간 표시
- 원형 프로그레스 링으로 달성률 시각화
- 기록 스와이프 삭제
- 새 도전 시작 (수동 초기화, 이전 기록 자동 보관)

## 실행 방법

1. `1000kmApp.xcodeproj`를 Xcode에서 열기
2. **Signing & Capabilities** 탭에서 개발자 팀 설정
3. 시뮬레이터 또는 실기기 선택 후 **Cmd+R** 실행

## 요구사항

- iOS 16.0+
- Xcode 14.0+
- Swift 5.0+

## 기술 스택

- SwiftUI
- MVVM 패턴
- UserDefaults (로컬 저장, 서버 불필요)
