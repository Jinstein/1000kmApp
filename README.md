# 1000km 걷기 챌린지 앱

1년 동안 누적 1000km 걷기 목표를 달성하기 위한 iOS 앱입니다.

## 기능

- 하루 걸은 거리(km) 입력 및 기록
- 누적 거리 및 남은 거리 실시간 표시
- 원형 프로그레스 링으로 달성률 시각화
- 기록 스와이프 삭제
- 새 도전 시작 (수동 초기화, 이전 기록 자동 보관)

## 시뮬레이터에서 실행

1. `1000kmApp.xcodeproj` 더블클릭 → Xcode 열기
2. 상단 기기 선택창에서 **iPhone 시뮬레이터** 선택
3. **Cmd+R** 또는 ▶ 버튼으로 실행

## 실제 아이폰에서 실행

### 1단계 - 아이폰 연결

USB 케이블로 Mac에 아이폰 연결 후, 아이폰에서 **"이 컴퓨터를 신뢰하겠습니까?"** 팝업이 뜨면 **신뢰** 선택

### 2단계 - 기기 선택

Xcode 상단 기기 선택창에서 연결된 **내 아이폰 이름** 선택

### 3단계 - 서명(Signing) 설정

1. 좌측 프로젝트 네비게이터에서 **1000kmApp** (파란 아이콘) 클릭
2. **Signing & Capabilities** 탭 클릭
3. **Team** 드롭다운에서 Apple ID 선택
   - Apple ID가 없으면 `Add an Account...`로 추가 (무료 계정으로도 설치 가능)
4. **Bundle Identifier**를 고유값으로 변경 (예: `com.홍길동.km1000app`)
   - 무료 계정은 Bundle ID가 다른 앱과 겹치면 오류 발생

### 4단계 - 아이폰 신뢰 설정

처음 설치 시 아이폰에서 아래 설정 필요:

**설정 → 일반 → VPN 및 기기 관리 → 개발자 앱 → 내 Apple ID → 신뢰**

### 5단계 - 실행

Xcode에서 **Cmd+R** → 아이폰에 앱이 설치되고 자동 실행됨

> **참고:** 무료 Apple ID로 설치한 앱은 **7일 후 만료**됩니다. 만료 시 Xcode에서 다시 Cmd+R로 재설치하면 됩니다. 만료 없이 사용하려면 **유료 Apple Developer Program** (연 $99) 가입이 필요합니다.

## 요구사항

- iOS 16.0+
- Xcode 14.0+
- Swift 5.0+
- Mac (Xcode는 macOS 전용)

## 기술 스택

- SwiftUI
- MVVM 패턴
- UserDefaults (로컬 저장, 서버 불필요)
